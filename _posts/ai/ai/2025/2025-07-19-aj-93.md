关于AGENTS.md使用

https://agents.md/ ，可以参考其它人的设置：https://github.com/search?q=path%3AAGENTS.md&type=code

1，放在根目录下

2，提供你想给新加入的同事的基本信息，如

Project overview

Build and test commands

Code style guidelines

Testing instructions

Security considerations

Commit messages or pull request guidelines

deployment steps
