CentOS Linux release 7.6 如何卸载php

rpm -qa | grep php
如果能看到 php-cli、php-common、php-fpm 等 rpm 包名，说明是 yum/rpm 安装。

yum remove php*


oniguruma5php-6.9.5+rev1-2.el7.remi.x86_64 是Oniguruma 正则表达式库

PHP 的 PCRE / mbstring 扩展依赖它，用于：

多字节字符串处理（mbstring）

正则表达式匹配


卸载mysql
yum remove mysql-community*

mysql57-community-release-el7-11.noarch MySQL 官方 Yum 仓库配置包（MySQL 5.7）作用只有一个：
在 /etc/yum.repos.d/ 里添加 MySQL 5.7 的 yum 源
只负责“告诉 yum 去哪里下载 MySQL 5.7”

│
├─ mysql57            → MySQL 5.7
├─ community-release  → 官方社区版 yum 源
├─ el7                → CentOS 7
├─ 11                 → 版本号
├─ noarch             → 架构无关
